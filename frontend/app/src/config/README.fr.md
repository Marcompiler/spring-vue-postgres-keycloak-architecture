# Frontend - config

## Description

Ici se trouve **la configuration du frontend**.

On trouve donc :

- La [**configuration de Keycloak**](./keycloak.js) : l'URL de l'instance de Keycloak avec le nom du realm et le client ID de l'application.
- La [**configuration du backend**](./backend.js) : l'URL menant vers l'API Spring.
